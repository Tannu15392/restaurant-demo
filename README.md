# 🍽️ Fireside Grill – Restaurant Website Demo

A modern **restaurant landing page** built using **HTML, CSS, and JavaScript**.

This project showcases a premium restaurant UI with smooth animations,
interactive menu filtering, a food gallery, and a reservation section.

---

## 🌐 Live Demo

https://tannu15392.github.io/restaurant-demo/

## ✨ Features

- Cinematic Hero Section
- Sticky Glass Navigation Bar
- Animated Menu Cards
- Menu Category Filtering
- Interactive Food Gallery
- Lightbox Image Viewer
- Reservation Call-To-Action
- Floating Order Button
- Smooth Scroll Animations
- Fully Responsive Design

- JavaScript
