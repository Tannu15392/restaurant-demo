

# 🍽️ Fireside Grill – Restaurant Website Demo

A modern **restaurant landing page** built using **HTML, CSS, and JavaScript**.
This project showcases a premium restaurant UI with smooth animations, interactive menu filtering, a food gallery, and a reservation section.

---

## 🌐 Live Demo

https://tannu15392.github.io/restaurant-demo/

---

## ✨ Features

- Cinematic Hero Section
- Sticky Glass Navigation Bar
- Animated Menu Cards
- Menu Category Filtering (Veg / Non-Veg / Dessert)
- Interactive Food Gallery
- Lightbox Image Viewer
- Restaurant Story Section
- Reservation Call-To-Action
- Floating Order Button
- Smooth Scroll Animations
- Fully Responsive Design

---

## 🛠️ Tech Stack

- HTML5
- CSS3
- JavaScript
- Google Fonts
- Unsplash Images
